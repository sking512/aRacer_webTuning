📖 系統參數說明與教學

⚙️ ECU 核心圖表 (VM / SA)

燃油 (VM - Volumetric Efficiency): 代表引擎在各轉速與負載下的容積效率。此數值決定了基礎的噴油時間，數值越大代表對應工況下需要的供油量越多。

點火 (SA - Spark Advance): 控制火星塞點火的提前角 (BTDC)。適當的提前能最大化扭力，但過早會引發爆震 (Knock)，過晚則會導致無力與排溫異常升高。

📊 負載參考基準 (AP / TPS)

AP (Absolute Pressure): 進氣歧管絕對壓力 -> 數值越高 (接近或大於大氣壓) 代表引擎負荷越重。適用於大多數平順行駛的供油/點火判定。

TPS (Throttle Position): 節氣門開度百分比 -> 在引擎大開度、換裝大凸輪軸或高轉速導致歧管壓力不穩定時，使用 TPS 作為 Y 軸參考能獲得更精準的油門反應。

🚀 介面操作教學

  • 雙表比對： 下方區分 Map A 與 Map B，可在 3D 圖表中直觀看出兩張圖表的差異。
  • 快速貼上： 支援直接從 aRacer 複製表格數據，點選下方表格格子後按下 Ctrl + V 快速覆蓋數據。
  • 誤差分析： 點擊右上角「誤差分析模式」，3D 圖將直接顯示 Map B 減去 Map A 的變化量 (Delta)。
  • 版面重置： 雙擊中間的「拖曳分隔線」即可讓版面回到置中狀態。
